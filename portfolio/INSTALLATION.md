## 📦 Installation & Running

Since the terminal seems to have PowerShell execution policy issues, please do this manually:

### Step 1: Open Terminal/Command Prompt
- Press `Ctrl + Shift + Terminal` in VS Code to open a new terminal
- Or use Windows Command Prompt or PowerShell with admin rights

### Step 2: Navigate to Portfolio Directory
```bash
cd "C:\Users\Abdul Rehmah\Desktop\Personal p\portfolio"
```

### Step 3: Install Dependencies
```bash
npm install
```

This will download and install all required packages (~300MB). It may take 2-5 minutes.

### Step 4: Start Development Server
```bash
npm run dev
```

The app will open at `http://localhost:5173`
